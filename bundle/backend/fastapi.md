FastAPI is a Python web framework built on type hints and Pydantic.
Request/response schemas are validated automatically from type
annotations, and it generates OpenAPI docs for free. Async-native, so
it handles concurrent I/O well despite being Python.

Written in [Python](../languages/python.md).

Best fit when: the team already has Python expertise (data science,
ML, automation scripts), needs to share code with ML/data pipelines,
or wants strong request validation with minimal boilerplate.
