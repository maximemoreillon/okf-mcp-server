React is a component-based UI library maintained by Meta. It has the
largest ecosystem of any frontend framework: component libraries,
state management options (Redux, Zustand, Jotai), and meta-frameworks
(Next.js) for routing, SSR, and data fetching.

Typically written in [TypeScript](../languages/typescript.md).

Best fit when: the team wants maximum hiring pool and third-party
library availability, or needs a mature meta-framework (Next.js) for
SSR/SSG.

See [Stack Pairing Guidance](../decisions/pairing.md) for how this
pairs with backend choices.
