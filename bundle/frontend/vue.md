Vue is a component-based UI framework with a gentler learning curve
than React and an official, cohesive set of tooling (Vue Router, Pinia,
Nuxt for meta-framework needs). Single-file components keep template,
script, and style together, which some teams find easier to reason
about than JSX.

Typically written in [TypeScript](../languages/typescript.md), though
Vue's template syntax stays readable in plain JavaScript too.

Best fit when: the team is smaller, wants an opinionated batteries-
included setup, or is onboarding developers less experienced with
JavaScript build tooling.

See [Stack Pairing Guidance](../decisions/pairing.md) for how this
pairs with backend choices.
